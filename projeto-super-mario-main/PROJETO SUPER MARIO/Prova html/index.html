Um dia, Mario e Luigi vão parar no reino dos cogumelos,
governardo pela Princesa Peach, mas ameaçado pelo rei dos
Koopas, Bowser, que vai fazer de tudo para conseguir reinar
todos os lugares. É então quando Luigi é raptado por
Bowser e o usa para procurar Mario, o unico capaz de deter
o Koopa e reestabelecer a paz.
